We Start in home directory : 

To install all about Git in linux just use : 
$ sudo apt-get install git

Then to connect your github you'll need a SSH key .
Use Command bellow to make new SSH key for this use : 
$ ssh-keygen

After running command above will see :
	
	Enter file in which to save the key (/home/ali/.ssh/id_rsa):

Just press "Enter" to generate the ssh key in the default path .

Then you'll see this : 
	
	Enter passphrase (empty for no passphrase):

Here can set a password for your git or just Enter for no pass !
After you'll see that : 

	Your identification has been saved in /home/ali/.ssh/id_rsa
	Your public key has been saved in /home/ali/.ssh/id_rsa.pub

Now to you need the public key to give it to repo settings
Now we are in home directory : 

$ cd ~/.ssh
$ cat id_rsa.pub

Then copy the output ssh key and follow the rest of instruction :

1) Go to the settings of your Git repo.
2) Go to the Deploy keys section.
3) Make a new key introducer by clicking  +Add Key.
4) Give a Name to your key for this repo.
5) Enable the write access.
5) Paste the public key here and add.

Then you need to configure your git with your identifications :

$ git config --global user.name "Your_Github_Username"
$ git config --global user.email "Your_Github_Email"

Now close the terminal and open again to use new git configuration .

Cangratulation !!
Now your are all set and can easily start your Gitting !! :)
